# Password Generator

A simple Python program that creates random passwords.

## Run

1. Install Python 3.8+.
2. Open a terminal in the project folder.
3. Run:

```bash
python passgen.py
```

## Options

- `-l`, `--length` ‚Äî password length
- `--no-upper` ‚Äî no uppercase letters
- `--no-lower` ‚Äî no lowercase letters
- `--no-digits` ‚Äî no digits
- `--no-symbols` ‚Äî no symbols
- `-n`, `--count` ‚Äî number of passwords
- `--cli` ‚Äî command-line mode

## Notes

- By default, the program opens a graphical interface.
- Use `python passgen.py --cli` to run in terminal.
