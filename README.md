# Password Generator

Generate secure random passwords with a simple GUI or command-line tool.

## Run

1. Install Python 3.8+.
2. Open a terminal in the project folder.
3. Run:

```bash
python passgen.py
```

## Options

- `-l`, `--length` — password length
- `--no-upper` — no uppercase letters
- `--no-lower` — no lowercase letters
- `--no-digits` — no digits
- `--no-symbols` — no symbols
- `-n`, `--count` — number of passwords
- `--cli` — command-line mode

## Notes

- By default, the program opens a graphical interface.
- Use `python passgen.py --cli` to run in terminal.
